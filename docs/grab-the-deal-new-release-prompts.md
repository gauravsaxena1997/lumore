## üí° Tips for Google Gemini Generation

### Image Generation (Gemini with Imagen 3):
- **Always attach reference images first** before submitting the prompt
- **Add explicit instruction:** "Match the exact style of the attached reference"
- **Be very specific** about Lumore branding, colors, and product details
- **Use exact hex codes** for colors in the prompt
- **Specify materials:** "frosted glass", "chrome metallic", "iridescent finish"
- **Mention lighting explicitly:** "soft ambient glow", "spot lighting", "rim lighting"
- **Request high resolution:** "1920x1080 minimum" or "high resolution render"
- **Aspect ratio:** Specify "16:9 landscape format"

### Getting Better Results:
- If first generation doesn't match reference well, add: "Follow the reference composition more closely"
- If colors are off, restate: "Use ONLY these Lumore brand colors: lilac #E5D4F7, ocean blue #B8D4E8, mint #D4E8DD, pearl white #FAFAFA"
- If products aren't clear, add: "All products must have visible 'LUMORE' branding and clear bottle shapes"
- Try generating 2-3 variations and pick the best one
- Use "photorealistic 3D render" and "luxury beauty commercial aesthetic" for style

### Video Generation (Google Veo):
- **Use the generated still image as input** for the video
- **Keep motion descriptions clear** and separate camera from object motion
- **Specify timing:** When each action happens in the 5-6 second duration
- **Request smooth motion:** "Professional, smooth, cinematic motion"
- **Avoid complex movements:** Focus on 1-2 primary motions
- **Request motion blur:** "Light motion blur for realism"
- **Maintain style:** "Keep the aesthetic and colors exactly as in the input image"

### Troubleshooting:
- **If style doesn't match reference:** Re-upload reference and emphasize "exact style match"
- **If wrong products appear:** List products explicitly with detailed descriptions
- **If colors are wrong:** Provide hex codes again and say "use ONLY these specific colors"
- **If composition is off:** Describe camera angle more specifically ("overhead 45-degree angle")
- **If quality is low:** Request "ultra-detailed, photorealistic, 8K quality render"
