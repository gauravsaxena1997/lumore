# Lumore Testimonial Video Prompts

**Purpose:** Customer testimonial videos for homepage
**Format:** Image-to-video conversion workflow
**Platform:** Google Veo 3 (or Luma Dream Machine / Kling AI as alternatives)
**Total Videos:** 7 (one per product SKU)
**Style:** Authentic product review aesthetic (d'you inspired)

---

## Workflow

1. **Generate Image** in Google Gemini/Imagen 3 using image prompt
2. **Convert to Video** in Veo 3 using image-to-video prompt
3. **Duration:** 5-8 seconds per video
4. **Format:** 9:16 portrait (mobile-optimized)
